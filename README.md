# Neung-Stock-AI
# 📈 AI Stock Analysis Dashboard
เครื่องมือวิเคราะห์แนวโน้มหุ้นด้วย AI สำหรับพอร์ตส่วนตัว (NVDA, AMD, VOO, VGT)

## 🚀 วิธีการใช้งาน
1. เข้าไปที่ลิงก์ Streamlit ของเรา
2. พิมพ์ชื่อ Ticker หุ้นในช่องด้านซ้าย
3. ดูการพยากรณ์ราคาจากเส้นประสีส้มในกราฟ

## 🛠 เทคโนโลยีที่ใช้
- **Language:** Python
- **AI Model:** Linear Regression (Scikit-learn)
- **Data Source:** Yahoo Finance
- **Interface:** Streamlit

## 📊 คู่มือการอ่านผลการวิเคราะห์ (Detailed Analysis Guide)

เพื่อให้การตัดสินใจลงทุนในหุ้นเทคโนโลยี (เช่น NVDA, AMD) หรือกองทุนดัชนี (VOO, VGT) มีประสิทธิภาพสูงสุด ควรพิจารณาผลลัพธ์ดังนี้:

### 1. การวิเคราะห์เส้นกราฟ (Chart Interpretation)
*   **เส้นทึบสีน้ำเงิน (Actual Price):** แสดงราคาปิดจริงในอดีต ใช้เพื่อดูความผันผวนย้อนหลังว่าหุ้นตัวนี้ "แกว่ง" แรงแค่ไหน
*   **เส้นประสีส้ม (AI Forecast):** คือแนวโน้มที่ AI คำนวณว่า "มีโอกาสเกิดขึ้น" หากสถิติเดิมยังดำเนินต่อไป
    *   **ชันขึ้นมาก:** หุ้นมีโมเมนตัมขาขึ้นที่แข็งแกร่ง
    *   **เส้นแบนราบ:** หุ้นอยู่ในช่วงพักตัวหรือรอเลือกทาง
    *   **ทิ่มลง:** สถิติบ่งชี้ว่าราคาเริ่มเข้าสู่ช่วงอ่อนแอ

---

### 2. การอ่านค่า Metrics (Key Performance Indicators)
*   **Target Price:** ราคาเป้าหมายเชิงสถิติที่ AI คาดการณ์ไว้ในอนาคต
*   **Percentage Change (%):** 
    *   **ค่าเป็นบวก (สีเขียว):** โอกาสทำกำไรตามแนวโน้มสถิติ
    *   **ค่าเป็นลบ (สีแดง):** ความเสี่ยงที่ราคาอาจย่อตัวลง

---

### 3. เทคนิคการปรับตั้งค่า (Optimization Tips)
*   **Short-term View (6mo):** เหมาะสำหรับการหาจังหวะซื้อ-ขาย (Trading) ในช่วงสั้นๆ
*   **Long-term View (1y - 2y):** เหมาะสำหรับนักลงทุนระยะยาว เพื่อดูภาพใหญ่ของหุ้น
*   **Tip:** หากเปลี่ยนช่วงเวลา (Period) แล้ว AI ให้ผลชี้ไปในทิศทางเดียวกันทั้งหมด จะเพิ่มความมั่นใจในการถือครองได้มากขึ้น

---
