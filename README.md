# Neung-Stock-AI
# 📈 AI Stock Analysis Dashboard
เครื่องมือวิเคราะห์แนวโน้มหุ้นด้วย AI สำหรับพอร์ตส่วนตัว (NVDA, AMD, VOO, VGT)

## 🚀 วิธีการใช้งาน
1. เข้าไปที่ลิงก์ Streamlit ของเรา
2. พิมพ์ชื่อ Ticker หุ้นในช่องด้านซ้าย
3. ดูการพยากรณ์ราคาจากเส้นประสีส้มในกราฟ

## 🛠 เทคโนโลยีที่ใช้
- **Language:** Python
- **AI Model:** Linear Regression (Scikit-learn)
- **Data Source:** Yahoo Finance
- **Interface:** Streamlit
